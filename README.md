# Flask Docker Ansible Project
A simple Flask web application deployed using Docker and Ansible on a local server.

## Project Overview

**This project demonstrates how to:**

1.Build a Python Flask web application

2.Containerize the application using Docker

3.Automate the deployment process using Ansible

**It is designed for Linux, DevOps, and beginner students to understand real-world deployment practices.**

## Technologies Used

   1.Python 3

   2.Flask

   3.Docker

   4.Ansible

   5.Linux (Ubuntu/Debian based system)

## Project Structure
---
flask-docker-ansible/

│   └── inventory           Ansible inventory
├── templates/
│   └── index.html          HTML page
└── README.md               Project documentation
---

## Features

1.Lightweight Flask web application

2.Dockerized application for portability

3.Ansible playbook for automated deployment

4.Runs on local server or cloud server

## How the Project Works

1.Flask application is created using Python

2.Docker packages the Flask app and dependencies

3.Ansible installs Docker and deploys the container

4.Application becomes accessible via browser

## Deployment Steps 

1.Clone the repository

2.Configure Ansible inventory

3.Run Ansible playbook

4.Access the application via browser

## Accessing the Application
Once deployed, open a browser and visit:
http://<server-ip>:<port>
